# react 脚手架(scaffold)


## 如何使用
1. 下载react的脚手架
```shell
npm install create-react-app -g
```
2. PWA (渐进性web app)

    可以用来配置(类似于一个原生APP)的方式,例如将一个网站作为一个移动端应用(
    例如我们经常将一个网站添加到桌面) 

    以及Service Worker(可以离线缓存),一般用不上 ..

    还能做消息推送 ..

    参考MDN 了解service worker的更多内容.

3. robots.txt

爬虫规定,那些能爬,那些不能爬..

4. 探索高级配置

这会将隐藏的配置暴露出来,不可逆 ..
